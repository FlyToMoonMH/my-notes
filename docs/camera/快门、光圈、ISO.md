# 光圈
大光圈，进光量多。
光圈还会对景深产生影响，光圈越小，背景虚化的效果越差。
# 快门
1/x，x越大，快门越慢。

快门速度越慢，光照射到传感器上时间越久。
如果要抓拍，就需要用很快的快门。
如果要拍摄一些风景，比如水面拉丝的效果，就可以用比较慢的快门，比如30s。

安全快门：1/焦距

# ISO
感光度，传感器对光线的敏感程度，作用就是牺牲
> ISO 调整的是位于传感器之后的模拟增益电路和模数转换器（ADC）的工作参数。

ISO 小，信号保持原样，细节丰富，但需要足够强的光。
ISO 越大、把微弱信号强行放大，暗部能看见了，但底噪也被放大。

有些相机在极低 ISO（如 ISO 64）时甚至会衰减信号来获得更大的动态范围。
