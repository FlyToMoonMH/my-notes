# 欢迎来到我的笔记站

这是一个为新手准备的个人网站模板。

你可以把它当成：

- 个人博客
- 技术笔记本
- 学习记录站
- 作品归档页

如果你现在还不懂 Linux、Git、HTML，也完全可以先用起来。

## 从哪里开始改

建议你先只改下面这 3 个文件：

- `docs/index.md`：首页介绍
- `docs/about.md`：关于我
- `docs/assets/data/site-data.json`：首页卡片内容

## 这个模板适合谁

- 想做一个像技术文档那样清爽的网站
- 希望主要用 Markdown 写内容
- 不想自己维护服务器
- 想用 GitHub Desktop 而不是记 Git 命令
