# Front Matter 模板

本站每篇 Markdown 文件建议在顶部添加如下元数据：

## 普通笔记

```yaml
---
description: 这篇笔记的主要内容摘要，会显示在搜索结果和社交分享中
tags:
  - tag1
  - tag2
comment: true        # 如需开启评论，添加此行
---
```

## 首页专用

```yaml
---
hide:
  - date
  - footer
home: true
template: home.html
statistics: true
---
```

## 不统计/不显示评论的页面

```yaml
---
nostatistics: true    # 不参与站点统计
comment: false        # 不显示评论区
---
```

## 常用字段说明

| 字段 | 作用 |
|------|------|
| `description` | 页面描述，用于 SEO 和社交分享 |
| `tags` | 标签列表，会出现在标签索引页中 |
| `comment` | 是否开启评论（需配置评论系统） |
| `nostatistics` | 设为 `true` 时该页不参与字数统计 |
| `hide` | 隐藏页面上某些元素，如 `date`、`footer`、`toc`、`navigation` |
| `title` | 覆盖页面标题（默认取第一个 H1） |
| `template` | 指定自定义模板，如 `home.html` |
